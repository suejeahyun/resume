4. AI 모델링 및 연동: 차량 번호판 및 핸드폰 번호 인식을 위한 AI 모델 통합.

5. 동적 페이지 구현: 사용자 입력 및 데이터 변경에 따라 실시간으로 업데이트되는 페이지 개발.

6. 알람 시스템: 최근 3시간 내 인식된 차량 및 이벤트를 표시하는 알람 페이지 구현.

7. 환경 설정: 관리자 페이지에서 요금 및 주차 관련 설정 가능하도록 구현.

8. 문의사항 관리: 관리자와 사용자별로 다른 페이지와 상태 추적 기능 제공.

9. 출퇴근 시스템: 출퇴근 시간 기록 및 주차 요금 자동 계산 기능 구현.

10. 차량 번호 검색: 특정 차량 번호를 검색할 수 있는 기능 구현.

11. 차량 데이터 조회: 입차/출차, 경차, 장애인 차량, 불법 주차 차량 조회 기능.

12. 웹캠 동적 생성: 웹캠 목록 관리 및 동적으로 생성하는 기능 구현.

총괄 역할:
- 백엔드 개발: 데이터베이스 설계, API 로직 구현, 사용자 인증 및 보안 체계 구축, AI 모델 연동.
- 프론트엔드 개발: 사용자 인터페이스 설계 및 동적 페이지 구현.
- 통합 작업: 백엔드와 프론트엔드를 연계하고 AI 모델을 웹 페이지에서 실행 가능하도록 구현.
